### Endpoints

- `GET /quotes`: Retrieves a list of quotes.
- `GET /quotes/{id}`: Retrieves a specific quote by ID.
- `POST /quotes`: Creates a new quote.
- `PUT /quotes/{id}`: Updates an existing quote.
- `DELETE /quotes/{id}`: Deletes a quote.
